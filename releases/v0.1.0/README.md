# NuiPet v0.1.0 发布说明

作者：暮逝的晚河（Wanhe）

NuiPet 是基于虚拟主播鹿弈Nui形象制作的轻量桌面宠物。当前发布包为 Windows x64 版本。

## 使用授权范围

本发布版本仅授权给虚拟主播鹿弈Nui的在舰或曾在舰的舰长使用。鹿弈Nui相关形象、名称、角色设定、素材与衍生视觉资产不随项目代码开源授权一并开放。

## 使用说明

请保持以下两个文件在同一目录中：

- `NuiPet-win_x64.exe`
- `resources.neu`

双击 `NuiPet-win_x64.exe` 启动桌宠。右键桌宠可打开菜单，切换动作、调整缩放、切换置顶或退出。左键点击桌宠会触发互动文本；拖动桌宠时会播放奔跑动画。
